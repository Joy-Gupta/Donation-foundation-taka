What is document object model (DOM)?

The Document Object Model (DOM) is a programming interface that represents the structure of a web page as a hierarchical tree of objects.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation in the Document Object Model (DOM) is a technique that allows you to handle events at a higher level in the DOM than the element that triggered the event.
